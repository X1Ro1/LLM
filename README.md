# API-пайплайн: Классификация отзывов через LLM

## Описание задачи
Скрипт автоматически классифицирует отзывы покупателей с помощью LLM модели, в моем случае использую Qwen. Для каждого отзыва определяется:
- Тональность: positive/negative/neutral
- Тема отзыва
- Уверенность модели в точности определения тональности отзыва
- Ключевые моменты, которые нейросеть отметила по комментарию

## Требования для установки перед началом работы:
- Node.js
- npm
Устанавить Node.js можно с официального сайта

## Установка и запуск
1. Для начала клонируйте репозиторий:
```bash
git clone https://github.com/X1Ro1/LLM.git
```
2. Далее открываете любую командную строку, в моем случае cmd и переходите в папку проекта:
```bash
cd LLM
```
3. Скачиваете зависимости npm
```bash
npm install
```
4. После установки зависимостей, можете запускать проект
```bash
npm start
```

После чего комментарии начнут отправляться нейросети для анализа. Результат находиться в новосозданном файле **analysis_res.json**.
