# API-пайплайн: Классификация отзывов через LLM

## Описание задачи
Скрипт автоматически классифицирует отзывы покупателей с помощью LLM, нейросеть - Qwen. Для каждого отзыва определяется:
- Тональность: positive/negative/neutral
- Тема отзыва
- Уверенность модели
- Ключевые моменты, которые нейросеть отметила по комментарию

## Требования для установки перед началом работы
- Node.js
- npm

## Установка и запуск
1. Клонируйте репозиторий:
```bash
git clone https://github.com/X1Ro1/LLM.git
```
2. Далее открываете командную строку, в моем случае cmd:
```bash
cd zad2
npm install
```

3. После установки зависимостей, можете запускать проект через npm
```bash
npm start
```

После чего комментарии начнут отправляться нейросети для обработки. Результат находиться в новосозданном файле - analysis_res.json
